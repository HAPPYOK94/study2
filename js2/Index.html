배열 심화

배열 판별

  1. Array.isArray()
    주어진 인자가 배열인지를 판별하여, 배열이라면 true를 아니라면 false를 반환한다.

배열생성

  1. Array.from()
    반복 가능한 객체 또는 유사 배열 객체를 복사해 새로운 배열 객체를 반환한다. 두번째 인자에 선택적으로 매핑할 함수를 지정할 수 있으며, 이 경우 주어진 배열을 복사할 때 각각의 요소에 지정된 함수를 실행하고, 그 결과를 배열로 반환한다.
  2. Array.of()
    3. of()는 인자의 개수나 데이터 타입에 무관하게 가변적인 인자를 가지는 새로운 배열을 반환한다.
  3. Array()와 Array.of()
    4. 여러개의 인자를 작성했을 때는 동일하게 동작 but 정수형 인자를 처리할때 Array(n)은 n개의 빈 요소를 가진 배열을 반환, Array.of(n)은 n을 요소로 가진 배열을 반환한다.

배열 요소 조작

  1. Push() / pop()
    push()는 배열 끝에 하나 이상의 요소를 추가하고, 변경된 배열의 길이를 반환한다.
    pop()배열의 끝에서 하나의 요소를 제거하고, 해당 요소를 반환한다.
  2. unshift() / shift() unshift()벼얼의 앞쪽에 하나 이상의 요소를 추가하고, 변경된 배열의 길이를 반환한다. shift()는 배열의 첫 번째 요소를 제거하고, 제거한 요소를 반환한다.

배열과 반복

  1. forEach()
    forEach()는 주어진 함수를 배열 요소 각각에 대해 실행하는 메서드로, 필수 인자로 실행할 함수가 요구된다. 또한 해당 함수(실행할 콜백 함수)는 필수 인자로 현재 요소(값)을 가지며, 선택적으로 현재 요소의 인덱스와 forEach()를 호출한 배열을 가질 수 있다.
    콜백 함수란 함수의 인자로 주어지는 함수를 의미
    forEach()는 원본 배열의 값을 변경하거나, 값을 리턴하지 않는다.

배열 요소 검색과 정렬

  1. inclues()
    includes()배열이 주어진 요소를 포함하는지 여부에 따라 true false값 반환
  2. find()
    주어진 판별함수를 만족하는 첫 번째 요소의 값을 반환한다. 이때 만약 반환할 값을 찾지 못한 경우 undefined를 반환한다.
  3. findIndex()
    주어진 판별 함수를 만족하는 첫번째 요소의 인덱스를 반환한다. 이때 만약 만족하는 요소가 없다면 -1을 반환한다.
  4. sort()
    배열을 정렬한 다음 (정렬된) 해당 배열을 반환한다. 이때 정렬의 순서는 기본적으로 문자열의 유니코드 순서에 따르기 때문에 숫자 정렬의 경우 정확하지 않을 수 있으며, 반환되는 배열은 새로운 배열이 아니라 원본 배열이 수정됨에 주의해야 한다.

배열 요소 수정

  1. fill()
    배열을 지정한 값으로 채우며, 원본 배열을 수정한다. fill()은 필수 인자인 배열을 채울 값과 선택 인자인 시작 인덱스, 끝 인덱스를 가진다.
  2. slice()
    배열의 복사본을 조각 내 새로운 배열로 반환하여, 선택적으로 시작 인덱스와 조욜 인덱스를 가질 수 있다.
  3. splice() 필수 인자로 시작 인덱스가 요구되며, 선택 인자로 배열에서 제거할 요소의 개수와 배열에 추가할 요소(들)을 가질 수 있다. 이때 splice()는 원본 배열 자체를 수정함에 주의해야한다.

기타 배열 매서드

  1. join()
    배열의 모든 요소를 이어붙여 하나의 문자열로 반환하며, 선택적으로 각각의 요소를 구분할 문자를 인자로 가질 수 있다. (생략할 경우 기본 구분자는 , 쉼표가 된다.)
  2. concat()
    인자로 주어진 배열(또는 값들)을 기존 배열에 합쳐 새로운 배열로 반환한다.
  3. map()
    배열의 모든 요소들에 각각 주어진 함수를 실행한 뒤 그 결과들을 모아 새로운 배열로 반환하며, 필수 인자로 실행할 함수가 요구된다. 또한 해당 함수(실행할 콜백 함수)는 필수 인자로 현재 요소(값)을 가지며, 선택
  4. filter() 주어진 판별 함수의 결괏값이 참인 요소들만을 모아 새로운 배열로 반환하며, 필수 인자로 실행할 함수가 요구된다. 또한 해당 함수(실행할 콜백 함수)는 필수 인자로 현재 요소값
  5. reduce()
